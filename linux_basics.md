# Linux Basics for DevOps

## What is Linux?
Linux is an open-source operating system widely used for servers, cloud, and DevOps environments.

## Why DevOps Engineers need Linux?
- Most production servers run on Linux
- Required for cloud, containers, and automation
- Helps in debugging and system management

## Basic Linux Commands
- pwd – show current directory
- ls – list files and folders
- cd – change directory
- mkdir – create directory
- touch – create file
- rm – remove file
- cp – copy file
- mv – move or rename file
- cat – display file contents
- echo – print/write text
- chmod – change file permissions

## Example
mkdir devops-day2  
cd devops-day2  
touch demo.txt  
echo "Day 2 Linux practice" > demo.txt  
cat demo.txt  

## Interview Notes
- Linux is preferred for servers due to stability and security
- chmod controls read, write, and execute permissions
