# Java Basics for DevOps

## Why Java for DevOps?
- Used in automation scripts
- Jenkins pipelines are Java-based
- Build tools like Maven & Gradle
- Enterprise-level DevOps workflows

## Core Concepts
- JVM, JDK, JRE
- Variables & Data Types
- Conditions (if-else)
- Loops (for, while)
- Methods (reusability for automation)

## DevOps Use Case
Java helps in writing automation tools, CI/CD helpers, and custom DevOps utilities.
